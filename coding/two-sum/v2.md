# Problem: Two Sum

### v2. Unique Pairs

#### Sources
- 

#### Description

Given an integer array `nums` and an integer `target`, find the number of in the array whose sum equals 
the `target`.

- Each pair should be counted only once.
- The order of elements in a pair does not matter (i.e., `[1][5]` and `[5][1]` are considered the same pair).
- Duplicate pairs must not be counted multiple times.

#### Examples

1. 
Input: `nums = [1][1][2][45][46][46]`, `target = 47`  
Output: `2`  
Explanation: The unique pairs are (1, 46) and (2, 45).

2. 
Input: `nums = [1][5][1][5]`, `target = 6`  
Output: `1`  
Explanation: (1, 5) is the only unique pair.

#### Constraints

- The array length $1 \leq \text{len}(nums) \leq 10^5$.
- Elements can be positive, negative, or zero.
- Target is an integer within the integer range.
- Efficient solutions with $O(n \log n)$ or $O(n)$ time complexity are expected.
- Space complexity ideally $O(n)$ or better.
