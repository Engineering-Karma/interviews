# Problem: Two Sum

***

### v3. Multiple Pairs

#### Sources
-

#### Description

Given an integer array `nums` and an integer `target`, find all pairs of numbers in the array whose sum equals 
the `target`. Unlike the unique pairs variation, here:

- You should return all pairs, including duplicates if they exist.
- The pairs can be returned in any order.
- Each pair consists of two elements whose sum is `target`.
- The same element may be used in multiple pairs but not twice in the same pair.

#### Example

Input: `nums = [1][5][1][5]`, `target = 6`  
Output: `[[1][5]`, `[1][5]]`  
Explanation: Two pairs sum to 6, both with elements 1 and 5.

#### Constraints

- Array length $1 \leq \text{len}(nums) \leq 10^5$.
- Elements can be positive, negative, or zero.
- Target is an integer within the integer range.
- Efficient solutions with $O(n)$ or $O(n log n)$ time complexity are expected.
- Space complexity ideally $O(n)$ or better.
